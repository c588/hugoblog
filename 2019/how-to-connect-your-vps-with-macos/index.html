在 MacOS 上连接 vps 服务器

入门 2019-12-15 450 字 1 分钟

　　自从装了黑苹果，发现连接服务器成了一个问题。之前使用 windows 10 只需要安装 xshell，现在发现 xshell 并没有 macOS 版本。在 bilibili 有位 up 主指出了使用 Royal TSX 可以完美代替 xshell，只是没有对应的汉化版本。但是在教程中使用的是 .pub 文件，蛋疼的是我使用的亚马逊 vps 提供的是另外一种——pem 文件。鼓捣半天总算连上了。鉴于网上找不到教程，这里把两种方法都记录一下。

第一种，macOS + Royal TSX + .pub：

Royal TSX SSH 密钥文件登录远程服务器

第二种，macOS + Royal TSX + .pem：

  1. 打开 Royal TSX

  2. 左上角 Royal TSX => plugins

  3. 下载 terminal 插件

  4. File => New Document

  5. 在创建的文件夹中选择 Connections

  6. 点击左下角的 + 号，选择 Terminal

  7. Computer Name 输入 vps 的 ip

  8. 点击左侧 Credentials

  9. 选择 Private Key File

  10. 默认选择 Path to Private Key File，切换到 Embed Private Key File

  11. 点击 Actions => browser，在弹出框内选择本地的 .pem 文件，open

  12. Apply & Close

  13. 双击出现的终端选项，填入登陆的用户名，回车

收工
