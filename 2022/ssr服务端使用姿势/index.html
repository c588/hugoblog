SSR服务端使用姿势

2022-12-27 406 字 1 分钟

CentOS 包管理工具是 yum，Debian 和 Ubuntu 是 apt

Centos 安装 SSR

  1. 购买 VPS

  2. 使用 ssh 连接 VPS。Windows 推荐 Termius，macOS 推荐 iterm

  3. 查看系统版本

    # 安装 lsb
sudo yum install -y redhat-lsb
lsb_release -a

    CentOS Linux release 7.6.1810 (Core)

  4. 安装 SSR

    # 下载脚本文件
wget --no-check-certificate -O [shadowsocks-all.sh](http://shadowsocks-all.sh/) [https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocks-all.sh](https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocks-all.sh)
# 设置运行权限
chmod +x [shadowsocks-all.sh](http://shadowsocks-all.sh/)
# 运行脚本
./shadowsocks-all.sh 2>&1 | tee shadowsocks-all.log

  5. 安装 BBR 加速

    # 下载脚本文件
wget --no-check-certificate -O [tcp.sh](http://tcp.sh/) [https://github.com/cx9208/Linux-NetSpeed/raw/master/tcp.sh](https://github.com/cx9208/Linux-NetSpeed/raw/master/tcp.sh) && chmod +x [tcp.sh](http://tcp.sh/) && ./tcp.sh
./tcp.sh
# 选择合适的加速方式，推荐原版 BBR 就够了
# 安装完成后重启 VPS，启用加速

  6. 安装 SSR 客户端

    • Windows
    • Android
    • MacOs
    • iPhone: 登录外区 App Store 后搜索 shadowrocket 下载安装使用

  7. 测速：

    # 下载脚本
wget [https://raw.github.com/sivel/speedtest-cli/master/speedtest.py](https://raw.github.com/sivel/speedtest-cli/master/speedtest.py)
# 添加权限
chmod a+rx [speedtest.py](http://speedtest.py/)
# 测速
python [speedtest.py](http://speedtest.py/)
# 选择节点
python speedtest.py --server 4751

Ubuntu 16 安装 V2ray 并启用锐速进行加速

  1. 设置 root 密码

    sudo passwd

  2. 下载并运行 233boy 的安装脚本

    sudo bash -c "$(curl -s -L https://git.io/v2ray.sh)"

    一路回车，保存最后显示的配置信息

  3. 安装锐速

    sudo v2ray

    选择 11

    选择 2 安装锐速加速

    内核不匹配

    选择1 安装 BBR 加速

  4. 重启 vps

  5. 先用 speedtest 脚本测速，下载脚本

    wget https://raw.github.com/sivel/speedtest-cli/master/speedtest.py

  6. 添加权限

    chmod a+rx speedtest.py

  7. 运行脚本测速

    python speedtest.py

    报错没有 python 命令，提示使用 python3 命令。测速完成：

    Hosted by Singtel (Singapore) [6.13 km]: 2.006 ms
Download: 485.25 Mbit/s
Upload: 276.00 Mbit/s

  8. 客户端启动并配置 v2ray
