《剑仙情缘三》四天观光有感

2020-04-25 3993 字 8 分钟

观看提醒

本片文章为个人在四天左右时间内，从创建角色到下场撕逼，体验到的一系列游戏内容，带主观因素，所有言论仅代表个人观点，请勿上纲上线。

全文艺术再创作，只为锻炼个人写作能力，文中出现所有人名、地名皆为虚构，如有雷同，纯属巧合。若您不信，关我屁事，我 TMD 还觉得复仇者联盟扯犊子呢。

因为相关政策原因，本站未能提供评论功能。如果感觉侵犯了个人权益，希望在贴吧指出。贴吧我看心情回复，这篇文章我看心情修改。这个就叫做话语权，划重点，下面会考。

竹杖芒鞋轻胜马，谁怕？一蓑烟雨任平生。

这里其实洋洋洒洒从某手贱党配台式机到 Windows —— Hackintosh —— Windows 的心路历程。写完我自己看着都要吐这又臭又长的语文作文都不带这么凑字数的所以索性删了。大概就是这娃头铁还憨，自个儿甚至可以跟自个儿较劲别说碰上心烦的人了，交代一下人物性格，做个铺垫。

我觉得要是都写就太啰嗦了，但是很多东西不写又怕被人指着说我哪哪刻意隐瞒了什么，不是小僧嘴太碎，实在是网友们太严格。算了，这里直接跳到进入游戏。

恍惚间经验渐满，惊喜中师傅到来

首先，就跟开始干活之前要过一遍开发文档一样，进入游戏之前，我在 WeGame 和剑网三贴吧稍微逛了一圈。我注意到有人说“大区攻防战排队一个多小时，推荐选人少一点的区，比如风月同天”。我的注意力是“排队”和“风月同天”。熟悉我的人都知道，我个人非常厌恶排队，一点点奶茶我从大学四年加上现在工作两年我就没买过几次，我一听到玩游戏要排队一个多小时我肯定溜。“风月同天”，疫情爆发，一线医疗防护资源告急，日本某县捐给武汉的物资上面贴着“山川异域，风月同天”，我看了新闻，所以一下子记住了。这就是我最后选择了剑三 WeGame 双线区，风月同天服务器。

职业选择了少林寺-少年，新手村升到三级，弹出一个推荐框，4800通宝，有一颗经验丹和一套2000品装备，恰好微信余额足够，平均一级不到五毛。升级后，又提示角色 18 级后必须充值游戏时长，否则无法获取经验，我又充值了 15 点卡。

人在成都。满级了干啥，不知道啊，根据网友，捡个师傅扫听扫听呗。于是世界频道喊话，捡到了 遇到了当前的师傅，“顾倾城的圣手”。 （Q：为什么进游戏三分钟就会用世界频道？A：CF、QQ三国、DNF 一路过来的，刷喇叭喊世界基本操作，虽然刚玩剑三，知道喊话没毛病吧？）

然后师傅告诉我要做 PVE 日常，PVP 日常和大小攻防。

涉世未深，两股战战入江湖；首战取胜，得意洋洋颇自负

// 大战截图

我在世界频道喊了几句，有没有人愿意带小白打副本。一开始有一个热心老哥【暮雪纷纷】密聊我，免费带我打，但是没注意消息。老哥2【圆不圆】告诉我，大晚上没人打副本，因为翻车就要熬夜，并且指导我怎么样使用团队招募功能参加到副本团中。

// 新手村热心老哥截图

// 25 人副本三张图

谁料到，一山还有一山高；不曾想，恶人更需恶人磨

老规矩，世界频道找团长。

// 万恶之源

13:46:08 之前，大概的内容是第一个 BOSS 过了，爆了很多道具，但是团长没有把分配方式改成主动分配，而是 Roll 色子，按点数大小自由分配。众所周知，自己能用的道具第一个按钮是“我要”，作为一个一穷二白的菜逼，我点得可顺手了，但是团长喊“大家点不要”以后我还是自觉地停住了。

// 摇号和梳理

// 纯粹的嘴臭，极致的享受

因为说了要把石头还给他们，所以我一直在副本入口等着，从 13:54 一直到 16:39 在贴吧“风月同天”发了类似帖子（多位吧友反应点不进去，但是我自己可以看到，大概率已被度娘屏蔽），历时2小时45分钟 。但是，确实，我被他们耍了。没有人出现，并且在后续他们口中，变成了 ”我在副本门口守着他们“ 。如果不是昨晚在监狱我被土拨鼠打死了五回，我都 TMD 信了他们的鬼话，我一个安装游戏 3 天的穷逼拖着一身 35000 品不到的装备蹲 一个团长 + 一个 PVE 帮派帮主 + （1-3？我也不确定）个帮众。

// 罚站图

// 土拨鼠反杀

妈的这段话越打越气，你们不仅说我黑你们一块钱，还说我充了 50（直升100级送2000品套装）+15（点卡）就变成氪金大佬可以去偷你们（3-5个人）的屁股了？你们是连我带着郭炜炜一起骂了是吧？

我只是在副本门口站着。人来人往，认识的，不认识的，蹦蹦跳跳的，轻功飞来的，都是来副本领装备的人。我多想拉一个团长，求求他：

[别信他们那些人了，行吗？秘境2天重置一次，我进去一会儿，立马被踢。为了凑点基础装备，BOSS CD没了，时间也被浪费了。刚刚看到一个团还没满，团长死活说我是小偷。那块石头值不值得偷，我自己心里没有数吗？一块六级石头才一块钱，我根本没必要去黑。谁家还不带个小白，你能保证一辈子玩游戏职业战队起步吗？啊？你们都不信我，我在野外只能等死。我不想苟着求生，我想站着挨打，行吗？]

// 行吗？

以上均为个人脑补。

实际情况就是 16:43 团长【明尊琉璃体】愿意带上我这个小白，继续打副本。

// 第三个团

还想狡辩，“我是我团成员！”；眼尖吧友，“长者吸取时间？”

相信不少吧友已经看到过，曾经贴吧中间有几层楼，是一个ID叫做【清X琬‬】的1.4年吧龄账号发布的，指着我装萌新，说自己是副本团队一员，当时团长在团里问了他十分钟他还不愿意交出石头如此如此。结果柯南附身的几位吧友直接指出逻辑不能自洽：你这加起来根本没有十分钟。最后在这几层回复在“不可控力”干扰的情况下被删除了。反正不是我啊。

其实是诈你们的，不考。

莺歌燕舞胡琴响，未闻远郊马啸声

就帮派活动啦，不多描述，聊天唱歌，我吹上天你们也不来加帮会。（若有意，欢迎联系风月同天恶人 PVP 帮会 倾尽天下 帮主【顾倾城的圣手】）

战火起，看似中立尽干坏事儿；帷幕落，虽属恶人共盼和平

帮派活动结束，我神行到蔷薇列岛恶人谷复活点，准备早上十点直接做日常，可以省下一个神行的 CD，虽然远比不上罗X祥出神入化的时间管理学，但我就是要称自己为风月同天不干正事儿小聪明。然后还没来得及下线呢：

卧槽这 TMD 有个自个儿帮会的虎逼朝我扔东西呢你是不是认错人了你这帮会名字跟我一样啊咱俩一伙儿的卧槽郭伟伟你把你家产品经理和策划一起开了把一个帮会还能内战你看美国队长获得的启发吗？

“后来啊···”我假装伸手去够木桌外沿的琥珀烟斗，实则只是为了活动一下一直没敢动的右手。抬眉瞅了一眼桌子对面摇椅上盖着毯子，呼吸依旧平稳的 迪丽热巴 刘亦菲 【】（先留空等我做好决定了再补），压低声音说：“后来山门下可是上来了一位了不得大人物。听说这片天地初开不久就有他活动过的痕迹，正反两派更是人脉、资源无数，连这个帮的帮主，都不得不敬畏三分。
