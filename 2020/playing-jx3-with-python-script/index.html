使用 Python 调用 DD 驱动玩剑三

2020-06-13 2490 字 5 分钟

前言

在家咸鱼了半个月后下载了《剑侠情缘网络版叁》这个游戏，体验一下几百人大乱斗的乐趣。陆续尝试了几个角色，发现可以使用 Python 来控制自动释放技能减少按键频率，在此记录一下实现过程。

只适用于阵营攻防战

环境

  • Windows 10 x64
  • python 3.8.4
  • DD 驱动 64 位免安装版
  • 剑侠情缘网络版叁（下称：剑网三）

思路

首先剑网三的角色技能分为三种：

  1. 第一类为瞬间释放，即没有技能前摇读条时间，也没有持续时间。
  2. 第二类为释放前需要读条。
  3. 第三类为技能释放前不需要读条但是有技能释放持续时间，在释放时无法移动。

技能释放读条不会被其他技能打断，需要持续释放的技能会被移动或者其他技能打断。所以，最理想的游戏角色为主要技能都是瞬间释放的第一类技能，其次是第二类，最后是第三类。根据对和这两个游戏职业技能的简单了解

【云裳心经（七秀）】：

云裳心经是一个治疗职业，主要技能为

  1. [繁音急节]：辅助技能，第一类，技能 CD 长，无公共 CD
  2. [余寒映日]：辅助技能，第一类，技能 CD 长，无公共 CD
  3. [风袖低昂]：大技能，第一类，技能 CD 长，有公共 CD
  4. [王母挥袂]：大技能，第一类，技能 CD 长，有公共 CD
  5. [上元点鬟]：小技能，第一类，无技能 CD，有公共 CD
  6. [翔鸾舞柳]：小技能，第一类，无技能 CD，有公共 CD
  7. [回雪飘摇]：小技能，第三类，无技能 CD，无公共 CD

技能循环：考虑到 1 和 2 为辅助类技能且无公共 CD，可以设置为每轮循环头部使用，即“有就放”“早用早 CD”。3 和 4 技能 CD 较长，可以在循环中部使用，也是“有就放”。5 和 6 因为没有技能 CD，在循环后部使用，降低优先级。2 3 4 5 四个技能在成功释放后会有 1.5 秒冷却，称为公共 CD（GCD）。7 技能在以上技能全部释放后瞬间释放。所以循环为 123457123467，即只要有 12 技能就必定释放，随后 34 技能 > 56 技能，最后补上 7 技能。7 技能释放时不可以移动或释放其他技能，否则会被打断。

目标选择：剑网三中在 设置 - 快捷键 中设置 Q 为选择自己，设置 E 为选择血量最低队友，释放技能前加上按 E 键这一操作即可。如果自身血线过低（40%），则疯狂按 Q 键，强制选择自己。

其他技能：三个减伤/解控技能手动释放，注意观察技能栏避免被卡 GCD 导致没有成功释放。

提高收益：12 技能手动释放，7 技能观察到读条打断手动补。

【天罗诡道（唐门）】：

天罗诡道是一个群攻职业，主要技能均不存在读条。

技能循环：千机变每隔 45 秒释放一次（降低位置移动导致离开攻击范围） > 千秋万劫 > 天绝地灭 > 暗藏杀机 > 天女散花

由于后四个技能都属于“有就放”的类型，此处不做具体释放顺序，而是根据优先级设置权重，减少被卡 GCD 的概率。经过按键测试，100次需要 0.00273s，完全可以忽略不计，只计算按键后的停留时间。因为技能 GCD 是 1.5s，为方便计算，每次按键后使用 sleep() 堵塞 30ms，50 次按键刚好 1.5s。50 次按键分别分配给后四个技能 25/15/4/4 次，和爆发技[心无旁骛] 1 次，[千机变]目标选择 1 次。循环前打乱顺序。

目标选择：每个攻击技能释放前均按下 Tab 键。

其他技能：[千机变]爆发技[鬼斧神工]、减伤免控技能 [惊鸿游龙] [飞星遁影]、辅助技[鸟翔碧空]、控制技[子母飞爪]等手动释放，注意观察技能栏避免被卡 GCD。[浮光掠影]技能必须关闭脚本停止自动释放技能后使用。

提高收益：手动释放爆发技和千机变，使用浮光掠影前关闭脚本。

代码实现：

为了实现开启脚本后自动切换到游戏界面，使用了 pywin32 这个库，需要提前安装。或者开启脚本后设置 1-3 秒延迟，手动切换。

使用 win32gui 库自动切换窗口时，只允许启动一个剑网三游戏进程。

早期剑三使用的是 SendInput API 来接受键盘和鼠标消息，那个时候任何一个按键工具都可以在剑三下生效，后来改用了 DirectInput API 来接受键盘和鼠标消息，DirectInput API 下的软件和游戏，是直接和键盘和鼠标的驱动层交互的，以前的按键方式已经无法在这种模式下工作了。

使用 DD 驱动 模拟按键。因为 DD 驱动免费版本在加载前会进行联网验证，有一定概率加载失败或加载时间过长，需要重新运行脚本。

查找并切换游戏窗口：

import win32gui

def get_all_hwnd(hwnd, mouse):
    if win32gui.IsWindow(hwnd) and win32gui.IsWindowEnabled(hwnd) and win32gui.IsWindowVisible(hwnd):
        hwnd_title.update({hwnd: win32gui.GetWindowText(hwnd)})
        
# 遍历窗口句柄，找到剑网三窗口并切换
hwnd_title = dict()
win32gui.EnumWindows(get_all_hwnd, 0)
jx3_exists = 0
for h, t in hwnd_title.items():
    if t != "":
        if t.find('剑网3 - ') >= 0:
            print('查找到窗口', h, t)
            hwnd = win32gui.FindWindow(None, t)
            if hwnd > 0:
                jx3_exists = 1
                left, top, right, bottom = win32gui.GetWindowRect(hwnd)
                print(left, top, right, bottom)
                win32gui.SetForegroundWindow(hwnd)
                print("窗口已切换")
                break
        else:
            print(h, t)

# auto_choose: 自动选择敌方单位  auto_buff: 自动开爆发
def tangmen(auto_choose=1, auto_buff=1):
    print("开始运行天罗诡道脚本")
    # 技能分配键位
    skills = {206: '[天女散花]', 207: '[暗藏杀机]', 208: '[千秋万劫]', 209: '[天绝地灭]',
              210: '[千机变]', 211: '[千机变-攻击]', 303: '[心无旁骛]'}
    for i in range(10000):
        if i % 30 == 0:
            orders = [210, 211, 208] * 20
            print("释放千机变")
        else:
            # 设置权重
            orders = [209] * 25 + [208] * 15 + [207] * 4 + [206] * 4 + [211]
            # 打乱技能散布
            random.shuffle(orders)
            if auto_buff:
                orders = [303] + orders
        start = time.time()
        if auto_choose == 1 and i % 2 == 0:
            dd_dll.DD_key(300, 1)
            dd_dll.DD_key(300, 2)
        for skill in orders:
            dd_dll.DD_key(skill, 1)
            dd_dll.DD_key(skill, 2)

        end = time.time()
