冒泡排序

对于 n 个元素的数组，第一轮通过 n-1 次比较可以确定最 大/小 的元素，第二轮通过 n-2 次比较可以确定第二 大/小 的元素。以此类推，第 n-2 轮通过 2 次比较可以确定第 n-2 个元素，第 n-1 轮通过 1 次比较确定第 n-1 （即倒数第二）个元素，最后一个元素不需要比较，就是最 小/大 的元素。关系：n 个元素的数组，需要 n-1 轮，第 i 轮需要比较 n-i 次。

<?php

function bub($arr) {
    $count = count($arr);
    
    if ($count < 1) return false;
    if ($count < 3) return $arr;
    
	for ($i = 0; $i < $count - 2; $i++) {
		for ($j = 0; $j < $count - $i - 2; $j++) {
			$k = $j + 1;
			if ($arr[$j] > $arr[$k]) {
				$arr[$j] += $arr[$k];
				$arr[$k] = $[$j] - $arr[$k];
				$arr[$j] -= $arr[$k];
			}
		}
	}
}

选择排序

跟冒泡排序类似，对于 n 个元素的数组需要进行 n 次遍历，第 i 次遍历确定第 i 个元素的位置。

<?php

function selSort($arr) {
    $count = count($arr);
    
    if ($count < 1) return false;
    if ($count < 3) return $arr;
    
    for ($i = 0; $i < $count - 1; $i++) {
        $min = $i;
        for ($j = $i + 1; $j < $count; $j++) {
            if ($arr[$j] < $arr[$min]) {
                $min = $j;
            }
        }
        if ($min != $i) {
            $arr[$i] += $arr[$min];
            $arr[$min] = $arr[$i] - $arr[$min];
            $arr[$i] -= $arr[$min];
        }
    }
    
    return $arr;
}

第二层循环结束后必须先判断当前最小元素的索引 $min 是否改变，如果仍为 $i 说明第一位就是最小数，不做元素调换。否则会赋值成 0。第一层循环结束条件为 $count - 1 因为需要循环 n - 1 次，第二层循环结束条件为 $count 因为 $j 已经赋值成 $i + 1。

类似的，如果需要循环 n 次，初始值为 0，条件为 < n，计数器从 0 到 n-1 执行 n 次。

快速排序

首先将数组中的第一个元素作为基准值，向后遍历，将小于基准值的数放入 $left 数组，大于基准值的数放入 $right 数组，将两个数组作为参数递归调用该方法，最后将 $left $right 数组以及 [$base] 结合，（按要求使用 sort/rsort）排序完成。

<?php

function quick($arr) {
    $count = count($arr);

    if($count <= 2) return $arr;

    $left = $right = [];
    $base = $arr[0];

    for($i = 1; $i < $count; $i++) {
        if($base > $arr[$i]) {
            $left[] = $arr[$i];
        } else {
            $right[] = $arr[$i];
        }
    }

    $left = quick($left);
    $right = quick($right);

    return array_merge($left, [$base], $right);
}

抢红包算法

漫画：如何实现抢红包算法？

  1. 二倍均值法
  2. 线段法

二倍均值法为了保证每个人分到的红包金额的数学期望相同，采用了在 (0, M/N * 2) 范围内取值。但是从微信的抢红包结果来看，有出现总额 300 的红包分成 150+、30+、10+ 的情况，而二倍均值法除了最后一位都不会超过 M/N * 2。

线段法随机性更强。例如总额为 100 的红包分给 10 个人，只需要在 (0, 100) 间确定 9 个随机点 n1~n9，很容易确定 10 个红包的金额为 n1, n2-n1, n3-n2, …, n9-n8, 100-n9 。

<?php
// red envelope

function redEnvelope($total, $count) {
	$total = $total * 100;// 转为分
	$envelope = $pos = [];
	do {
		$pos[] = mt_rand(1, $total - 1);
	} while (count($pos) != $count - 1);
	sort($pos);
	print_r($pos);
	$envelope[] = $pos[0] / 100;
	foreach ($pos as $key => $value) {
		if ($key == 0) continue;
		if ($key == $count - 2) break;
		$envelope[] = ($pos[$key + 1] - $value) / 100;
	}
	$envelope[] = ($total - $value) / 100;
	return $envelope;
}

主要需要处理的地方为生成随机点时，需要确定范围，最小值为 1 分，所以应该是 [1, $total * 100 - 1]，随机函数使用 mb_rand()，官方说明：生成更好的随机数。mt_rand() 函数是非正式用来替换 rand() 。

猴子选大王问题

即约瑟夫环。共 M 只猴子，编号 1-M，依次排数，当数到为 N 时出局，继续从 1 开始。循环至最后一只猴子停止，求编号。

大一 C 语言课程题目，小白噩梦。

直观解法：用数组储存所有猴子编号，for 循环计数，如果序号不为 N 的倍数，将当前元素放到数组末尾，否则删除当前元素。当数组内元素个数为 1 时，输出该元素。

<?php
// joseph ring

function monkeyKing($m, $n) {
    $arr = range(1, $m);
    $i = 0;
    while(count($arr) > 1) {
        if(($i + 1) % $n) {
            $arr[] = $arr[$i];
        }
        unset($arr[$i]);
        $i++;
    }

    return $arr[$i];
}

print_r(monkeyKing(5000, 5000));  // 3977
print_r("\n");

缺点：时间复杂度高达O(nm)

数学解法： 没看懂 约瑟夫环——公式法（递推公式）

n阶井字棋

井字棋，国外称为 Tic-Tac-Toe ，规则为“两个玩家，一个打圈（◯），一个打叉（✗），轮流在3乘3的格上打自己的符号，最先以横、直、斜连成一线则为胜。”现在要求代码实现对 n 阶井字棋的胜负判定。

每下一步棋，进行 2+1+1 次判断，横向是否一致，纵向是否一致，如果在任意一条对角线上，额外进行 1 或 2 次对角线判断。

<?php

$n = 5;  // 五阶棋盘
$res = array();

function check($arr)
{
    $n = $GLOBALS['n'];
    $res = $GLOBALS['res'];
    // 已经下过这一步，返回false；没有，赋值
    if (isset($res[$arr[1]][$arr[2]])) {
        return false;
    } else {
        $res[$arr[1]][$arr[2]] = $arr[0];
    }
    // 如果某一步下在对角线上，额外进行一次对角线判断(前提：阶数为奇数)  // 错误，阶数为偶数也会出现对角线
    //if ($n % 2 == 1) {
    if (1) {
        if ($arr[1] == $arr[2]) {
            for ($i = 0; $i < $n - 1; $i++) {
                if (!(isset($res[$i][$i]) && isset($res[$i + 1][$i + 1])) || $res[$i][$i] != $res[$i + 1][$i + 1]) {
                    break;
                } elseif ($n - 2 == $i) {
                    var_dump($arr[0] . '--- Win!');
                    $GLOBALS['res'] = $res;
                    return true;
                }
            }
        } elseif ($arr[1] + $arr[2] + 1 == $n) {
            for ($i = 0; $i < $n - 1; $i++) {
                if (!(isset($res[$i][$n - 1 - $i]) && isset($res[$i + 1][$n - 2 - $i])) || $res[$i][$n - 1 - $i] != $res[$i + 1][$n - 2 - $i]) {
                    break;
                } elseif ($n - 2 == $i) {
                    var_dump($arr[0] . '--- Win!');
                    $GLOBALS['res'] = $res;
                    return true;
                }
            }
        }
    }
    for ($i = 0; $i < $n - 1; $i++) {
        // 横列判断(是否赋值 or 是否相等)，不符合则进行纵列判断
        if (!(isset($res[$arr[1]][$i]) && isset($res[$arr[1]][$i+1])) || $res[$arr[1][$i]] != $res[$arr[1][$i+1]]) {
            // 进行纵列判断(是否赋值 or 是否相等)，不符合则跳出两层循环
            for ($j = 0; $j < $n -1; $j++) {
                if (!(isset($res[$j][$arr[2]]) && isset($res[$j+1][$arr[2]])) || $res[$j][$arr[2]] != $res[$j+1][$arr[2]]) {
                    break 2;
                } elseif ($n - 2 == $j) {
                    var_dump($arr[0] . '--- Win!');
                    $GLOBALS['res'] = $res;
                    return true;
                }
            }
            break;
        } elseif ($n - 2 == $i) {
            var_dump($arr[0] . '--- Win!');
            $GLOBALS['res'] = $res;
            return true;
        }
    }
    $GLOBALS['res'] = $res;
}

字符串处理

给定字符串 “Hello WORld” 要求转换成 “Olleh DLRow”，逆序，对应位置大小写不变。

字符串逆序，按照原字符串改变大小写。

<?php

function strFmt($str) {
    $len = strlen($str);
    $res = strrev($str);
    for ($i = 0; $i < $len; $i++) {
        if (ord($str[$i]) > 64 && ord($str[$i]) < 91) {
            $res[$i] = strtoupper($res[$i]);
        }
        if (ord($str[$i]) > 96 && ord($str[$i]) < 123) {
            $res[$i] = strtolower($res[$i]);
        }
    }

    return $res;
}

ord() 方法可以解析 string 二进制值第一个字节为 0 到 255 范围的无符号整型类型。此处返回的是字母在 ASCII 编码表中的位置。

ASCII 表中，65-90 对应大写字母，97-122 对应小写字母。
